# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a philosophy text collection repository containing excerpts and essays from various philosophical works and authors. The repository is organized into thematic directories containing plain text files.

## Repository Structure

The repository contains the following main directories:

- **A-Z/**: Alphabetical reference materials
- **ALHoP/**: "A Little History of Philosophy" by Nigel Warburton - 40 chapters covering major philosophers from Socrates to Peter Singer
- **PB/**: "Philosophy Bites" - 25 essays on various philosophical topics (ethics, metaphysics, aesthetics, religion)
- **PBA/**: "Philosophy Bites Again" - 27 additional essays continuing the Philosophy Bites series
- **PBB/**: "Philosophy Bites Back" - 21+ essays focusing on historical philosophers and their key works
- **PTB/**: "Philosophy: The Basics" - 2 sections on fundamental topics (God, Right and Wrong)
- **PTC/**: "Philosophy: The Classics" - 32 chapters covering major philosophical texts from Plato's Republic to Rawls' Theory of Justice
- **TR/**: "The Republic" by Plato - Organized into 10 books

## File Naming Conventions

Files follow these patterns:
- Numbered entries: `1. AUTHOR ON Topic` or `1. Author Name`
- No file extensions (plain text files)
- Titles use various capitalization styles depending on the source collection

## Working with This Repository

### Searching for Content

When searching for philosophical topics, authors, or concepts:
- Use `Grep` to search across all files for specific terms or philosophers
- Use `Glob` with patterns like `**/[0-9]*` to find numbered entries
- Directory names indicate the source collection

### Reading Files

- All files are plain text without extensions
- Files contain dialogue-style philosophical discussions or essay-style content
- Content is formatted with line breaks and paragraph spacing
